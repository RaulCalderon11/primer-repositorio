Introducción

La asignatura de Desarrollo de Aplicaciones Móviles Web es la base para que los estudiantes adquieran las habilidades necesarias para crear soluciones web. Nos enfocamos en la experiencia de usuario y el diseño moderno, utilizando las herramientas y tecnologías más actuales del mercado.

Objetivos

  • Conocer los conceptos fundamentales del desarrollo web.
  • Aprender a diseñar y desarrollar interfaces de usuario.
  • Entender la importancia de la experiencia del usuario.
  • Desarrollar habilidades para resolver problemas y depurar código en entornos de desarrollo integrados (IDEs).

Contenidos

  • Introducción al desarrollo web.
  • Lenguajes de programación para web (HTML, CSS, JavaScript).
  • Frameworks para desarrollo web (React, Vue, Angular).
  • Diseño de interfaces de usuario para aplicaciones web.
  • Desarrollo de aplicaciones web utilizando APIs y bases de datos.
  • Pruebas y depuración de aplicaciones web.
  • Publicación y distribución de aplicaciones web.
